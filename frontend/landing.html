Transparency

Local-first, not vague-first.

AudioEnhancerMAX is built for governance-sensitive workflows: data movement is explicit, AI suggestions are assistive, and original media stays available for review.

Local Core Pipeline

Uploads, DSP processing, Whisper transcription, Kokoro TTS, timing history, and Ollama/Gemma inference run on the host machine when those local dependencies are installed.

Local-first

Optional External Services

Edge Neural TTS sends selected text to Microsoft when that engine is used. The public SourceForge badge loads a SourceForge script in the browser.

Explicit opt-in

Explainability Boundaries

Selected filters, preset options, ETA confidence, and Smart Mode suggestions are visible. Audio quality and transcript accuracy still require human review for high-stakes use.

Reviewable
Features

Processing modules
you can inspect.

Each module maps to a visible option or preset. The app favors concrete controls over hidden automatic edits.

Noise Removal

DeepFilterNet v3 when available, plus spectral gating and wet/dry mixing. Designed to reduce artifacts while preserving voice texture.

DeepFilterNet

Filler Word Removal

Whisper-based timestamps help identify common filler patterns such as "uhm", "eh", and "basically" for reviewable cleanup.

Whisper AI

Studio Sound v2.0

Voice-focused chain with warmth at 150Hz, presence around 4kHz, de-essing, gentle 2:1 compression, and limiting.

Pedalboard

Metal GPU Acceleration

Demucs music separation runs on Apple Metal GPU via PyTorch MPS. ARM NEON optimized CTranslate2 for Whisper inference.

Metal MPS

Ollama/Gemma Smart Mode

When Ollama/Gemma is configured locally, Smart Mode suggests a content type and processing settings. Suggestions remain reviewable.

Local Ollama

Edge Cluster Computing

Turn trusted Android devices into LAN workers for selected DSP tasks. Discovery and manual registration stay under user control.

UDP Discovery

Music Preservation

Meta's Demucs separates vocals from music. Process speech only, remix with original background music intact.

Demucs + Metal

Real-time Monitoring

Per-core CPU heatmap, GPU usage, power watts, ANE status. Processing dashboard with live pipeline visualization.

psutil + macmon

AI Transcription

Word-level timestamps with export to SRT, VTT, and JSON. Speaker diarization uses pyannote.audio when available, with fallback behavior.

Whisper + pyannote

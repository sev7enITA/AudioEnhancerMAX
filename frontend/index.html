🏗️ Architecture

🖥️ Backend FastAPI + Python 3.11
🎨 Frontend Vanilla JS + CSS3
🧠 AI Engine Whisper + Ollama/Gemma
⬡ GPU PyTorch MPS (Metal)
📱 Edge Workers Android Kotlin + HTTP
🔊 Audio DSP scipy + pedalboard

🆕 What's New in v3.5.1

NEW Streaming transcription — see text appear in real-time as Whisper processes
NEW Non-blocking server — HW metrics stay live during heavy processing
NEW Incremental file saving — crash-resilient transcripts saved segment-by-segment
NEW Adaptive ETA engine — pre-task estimates with confidence badges
NEW Custom output filename — choose your transcript filename before processing
FIX Settings & About panels now render correctly (CSS specificity fix)
FIX System monitor v2 — separate threads for psutil + macmon, proper warmup
FIX HW polling active for ALL task types (not just Audio Processing)

📋 v3.0 Features

NEW Apple Metal GPU acceleration (PyTorch MPS) for Demucs music separation
NEW Distributed Edge Cluster — offload DSP filters to Android smartphones
NEW Real-time processing dashboard with per-core CPU heatmap
NEW Built-in DSP benchmark for cross-device performance comparison
NEW Apple Silicon acceleration module (Accelerate vDSP + ARM NEON + MPS)
IMPROVED Dynamic filter tuning via local Ollama/Gemma inference
IMPROVED v2.0 studio sound chain — broadcast-quality compression + de-esser
IMPROVED Noise removal with wet/dry mix to prevent metallic artifacts
